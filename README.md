# Project Overview
The **Alpha Project**  is an academic project created using **Monday.com** to demonstrate structured project management and data-driven monitoring of business workflows.  
The project focuses on planning and execution phases, tracking tasks, deadlines, priorities, budgets, and team responsibilities in a centralized system.

This project highlights how operational project data can be analyzed through dashboards, metrics, and automation to support informed decision-making.

## Objectives
- Organize and track project tasks using a structured board
- Monitor progress using real-time dashboards and KPIs
- Identify delays, risks, and workload distribution
- Apply automation to improve efficiency and accountability
- Demonstrate business and operational analytics concepts

## Key Features & Analytics
 **Task Tracking:** Status, priority, due dates, timelines, and budget columns  
 **Dashboards:**  
   Task completion distribution (Done / In Progress / Stuck)  
   Overdue task analysis  
   Workload distribution by owner  
 **Metrics & KPIs:**  
   Percentage of completed tasks  
   Overdue and at-risk tasks  
 **Automation:**  
   Deadline reminder notifications  
   Automatic escalation of priority to *Critical* for overdue tasks  
**Audit & Logs:**  
   Automation run history with timestamps and success tracking
